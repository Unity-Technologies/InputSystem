Detailed Description

Turns binary HID descriptors into HID.HIDDeviceDescriptor instances.

For information about the format, see the Device Class Definition for Human Interface Devices section 6.2.2.

Parse a HID report descriptor as defined by section 6.2.2 of the HID specification and add the elements and collections from the descriptor to the given deviceDescriptor .

Parameters
bufferBuffer containing raw HID report descriptor.
deviceDescriptorHID device descriptor to complete with the information from the report descriptor. Elements and collections will get added to this descriptor.
Returns
True if the report descriptor was successfully parsed.

Will also set HID.HIDDeviceDescriptor.inputReportSize, HID.HIDDeviceDescriptor.outputReportSize, and HID.HIDDeviceDescriptor.featureReportSize.
